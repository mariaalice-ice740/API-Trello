# 📌 Trello API – Postman Tests

## 📖 Sobre

Coleção de testes de API desenvolvida no Postman utilizando a API do Trello.

O foco é validar operações de boards, listas e cards, garantindo o funcionamento correto dos fluxos principais da aplicação.

---

## 🚀 Tecnologias

* Postman
* JavaScript (scripts de teste)

---

## ✅ Cenários testados

* Criar, buscar e deletar boards
* Criar listas (TODO / DONE)
* Criar e consultar cards
* Mover card entre listas

---

## 🔄 Fluxo principal

Board → Lists → Card → Move → Delete

---

## ▶️ Como executar

1. Importar a collection no Postman
2. Configurar `api_key` e `token`
3. Executar via Collection Runner

---

## 📌 Aprendizados

* Testes de API no Postman
* Uso de variáveis e encadeamento de requisições
* Validação de respostas com JavaScript
